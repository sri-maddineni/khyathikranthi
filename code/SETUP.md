# Khyathi Associates Website - Setup Instructions

## Installation

Before running the development server, you need to install the required dependencies:

```bash
npm install react-icons
```

## Running the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

## Project Structure

- **Home Page** (`/`) - Main landing page with hero section and overview
- **About Page** (`/about`) - Information about Khyathi Associates
- **Cases Page** (`/cases`) - Detailed list of practice areas and cases dealt
- **Profile Page** (`/profile`) - Professional profile of Kranthi Naidu Boya
- **Contact Page** (`/contact`) - Contact information and contact form
